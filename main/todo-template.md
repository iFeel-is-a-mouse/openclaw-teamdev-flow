# todo.md — 项目任务看板

> **阶段结束标志：该阶段所有 todo 项全部 ✅。**
> 5 个 agent 均可编辑。修改/删除/削弱任何条目需用户最高层同意。
>
> 使用方法：复制此模板到 `projects/<项目名>/docs/todo.md`，按项目实际流程裁剪。

---

## 项目信息

| 字段 | 值 |
|------|-----|
| 项目名称 | `<项目名>` |
| 复杂度 | S / M / L |
| 流程阶段数 | `<N>` |
| 启动时间 | `YYYY-MM-DD HH:mm` |
