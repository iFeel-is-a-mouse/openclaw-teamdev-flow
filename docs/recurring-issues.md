# 重复性问题追踪

> 记录已出现 ≥ 2 次的重复性问题。每次出现都加深一层根因分析，直到找到流程级修复方案。
>
> 管理规则见 `docs/change-management.md §11.3`。
>
> **第三次出现时**必须做流程级修复（加门禁、建自动化检查），不只是修改文档。
